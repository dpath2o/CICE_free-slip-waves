## 2) Where I left this (end of Oct 2025)

### 2.1 Free-slip boundary condition: discrete formulation drafted + idealised sanity tests
I wrote up a concrete discrete strategy for **no-slip (Dirichlet)** vs **free-slip (Neumann)** handling of velocity values that appear in *finite-difference stencils* near the land boundary.

Key points:
- **Impermeability is always enforced** via masks on the *normal* flow at solid faces:
  - `u_E <- m_E * u_E`
  - `v_N <- m_N * v_N`
- The remaining question is how to treat **masked neighbours in derivative/strain-rate stencils**.
  - **No-slip (Dirichlet):** if neighbour lies on land, clamp neighbour to `0`.
  - **Free-slip (Neumann):** if neighbour lies on land, set neighbour equal to local value (even reflection), so the discrete normal gradient vanishes.

This was written down in a simple 2×2 C-grid schematic (T/E/N/U points) with the “effective neighbour” recipe expressed using face masks (`m_E`, `m_N`).

I also attached idealised animations comparing free-slip vs no-slip behaviour for a **uniform eastward wind forcing** case on a small uniform grid (pre-coastal-drag).

**Open design detail to confirm with you:**
- Whether we want free-slip to apply only to *tangential components* in the relevant stencils (with normal components already handled by impermeability masking), or whether we keep the generic “effective neighbour” logic as written for any `q` used in the stencil.

## 5) Appendix note I’ve added: “Motivation and Regridding Strategy” (fast-ice diagnostic)
Because the **B-grid no-slip** in CICE standalone creates corner-based “dead zones” (speed artefacts) adjacent to land, I clarified the speed diagnostic used for landfast-ice classification:

- Compute native B-grid speed magnitude: `|u_B| = sqrt(u_ice^2 + v_ice^2)`.
- Build T-grid speed options from (u,v):
  - **Ta:** simple 2×2 box-average of B-grid corners around each T-cell.
  - **Tb:** a no-slip–aware 2×2 average that treats land-corner values as zero *before* averaging (denominator still 4).
  - **Tx:** bilinear B→T regridding via precomputed ESMF weights, then magnitude.

For the experiments in the paper, I’m using **Tb exclusively** as the fast-ice speed diagnostic:
- `A_ice > A_thresh` and `0 < |u_Tb| <= u_thresh`

Rationale:
- Tb suppresses the unphysical “dead” T-cells right next to land that arise from corner masking,
- while still respecting the intended no-slip behaviour and co-locating speed with T-grid concentration.
